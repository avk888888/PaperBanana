AutoFigure

Generating and Refining Publication-Ready Scientific Illustrations

ICLR 2026

GitHub Paper Website Dataset

About

AutoFigure is an intelligent system that leverages Large Language Models (LLMs) with iterative refinement to generate high-quality scientific figures from text descriptions or research papers. From text to publication-ready diagrams.

Features

📝 Text-to-Figure

Generate figures directly from natural language descriptions.

📄 Paper-to-Figure

Extract methodology from PDFs and create visual diagrams.

🔄 Iterative Refinement

Dual-agent system for continuous quality optimization.

🎨 Multiple Formats

Output as SVG or mxGraph XML (compatible with draw.io).

💅 Image Enhancement

AI-powered post-processing for aesthetic beautification.

🖥️ Web Interface

Interactive Next.js frontend for easy generation.

How It Works

AutoFigure employs a Review-Refine loop: (1) Generate — creates SVG/XML from description; (2) Evaluate — scores quality and provides feedback; (3) Refine — continues until publication standards are met.

Quick Start

git clone https://github.com/ResearAI/AutoFigure.git
cd AutoFigure
pip install -e .
playwright install chromium

# Python usage:
from autofigure import AutoFigureAgent, Config
config = Config(generation_api_key="your-api-key")
agent = AutoFigureAgent(config)
result = agent.generate("Transformer architecture diagram")
result.save("output.svg")
